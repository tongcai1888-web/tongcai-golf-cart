# TONGCAI Golf Cart Independent Website

Static B2B website package for GitHub Pages deployment.

## Included
- Long-scroll homepage with approved TONGCAI visual direction
- Products catalog and product detail pages
- Customization, Factory, Solutions, Dealer, About and Contact pages
- Official TONGCAI logo, banner images and product images
- CSS and JavaScript assets
- robots.txt and sitemap.xml

## Deploy to GitHub Pages
1. Create a GitHub repository.
2. Upload all files in this folder to the repository root.
3. Go to Settings -> Pages.
4. Choose Deploy from branch, select `main` and `/root`.
5. Update `robots.txt` and `sitemap.xml` with your real domain.

## Before launch
- Verify company address, phone, email and factory data.
- Replace placeholder factory/solution images with real photos when available.
- Connect the inquiry form to Formspree, Tally, Basin, or a custom backend if you want automatic lead collection. Current forms open the user's email client.
